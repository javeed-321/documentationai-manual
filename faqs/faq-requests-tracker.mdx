## Overview

This page tracks questions submitted for the FAQ. Use it to see whether your question is already in the queue and where it stands. Answered items graduate to the [Frequently Asked Questions](/faqs/frequently-asked-questions) page.

<Callout kind="info" collapsed="false">
  This tracker is a documentation showcase placeholder. Connect it to your real intake form or ticket queue before publishing.
</Callout>

## Request status reference

| Status         | Meaning                              |
| -------------- | ------------------------------------ |
| Submitted      | Received and waiting for triage      |
| Being answered | An owner is drafting the answer      |
| Published      | Live on the FAQ page                 |
| Declined       | Out of scope, with a reason recorded |

<Callout kind="tip" collapsed="false">
  Popular requests are prioritized. If a question matters to your team, submit it even if it looks similar to an existing one; volume is a useful signal.
</Callout>
